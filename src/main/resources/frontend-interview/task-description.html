Welcome to our Frontend Developer Task

Why a programming task?

In order to get an impression of your coding skills we are asking you to do what you do best - write some code. Solving the following task will give us a basic understanding of your style and will be used as a basis for further questions and open technical discussion. There is no right or wrong solution, we just want to see how you will approach the problem.

The Task

Create an angular file manager application with the following features:

  • The file manager shows available images
  • Each image must be represented by a thumbnail, as well as it's name and size
  • The file manager shall include a file upload area, that is triggered either by a button (standard behavior) or directly by dran-and-dropping files
  • Multiple file upload shall be possible
  • File upload progress is shown in some way (numbers, percentage, progress bar, ...)
  • File upload status is visible (error or success)

Technical Requirements

  • Angular 14+
  • UI library of your choice (or custom CSS)

Instructions

Getting data about files

To get a list of files, you can use one the following endpoints

  • /api/v1/files - gets a list of all files
  • /api/v1/files/{id} - gets a list of all articles separated in pages

To get a thumbnail of a desired size, you can use the parameter size on the /api/v1/files/{id}. For example size=100 will return a thumbnail of max size 100x100px.
